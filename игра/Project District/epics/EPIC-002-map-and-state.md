# EPIC-002: Карта и состояние мира

## Description

Создать фиксированную карту, стартовые банды, бригады и параметры районов.

## Stories

### STORY-002-001: Четыре района
**User Story**: As a игрок, I want to видеть 4 района so that понимать поле конфликта.
**Acceptance Criteria**:
- [ ] Есть Барная улица, Рынок, Склады, Старые дома.
- [ ] У каждого района есть доход, защита, жара, напряжение.
- [ ] Соседства соответствуют схеме 2x2.
**Size**: S

### STORY-002-002: Стартовые фракции
**User Story**: As a игрок, I want to иметь конкурентов so that районы были спорными.
**Acceptance Criteria**:
- [ ] Есть банда игрока, Южные, Крысы.
- [ ] У каждой банды есть деньги, сила, агрессия, осторожность.
- [ ] Владение районами задано на старте.
**Size**: S

### STORY-002-003: Две бригады игрока
**User Story**: As a игрок, I want to выбирать между бригадами so that решение "кого отправить" имело смысл.
**Acceptance Criteria**:
- [ ] Есть две бригады с разными параметрами.
- [ ] Состояние бригады влияет на доступность приказов.
- [ ] Параметры видны игроку.
**Size**: M
