# EPIC-004: Отчеты и обратная связь

## Description

Сделать отчеты главным экраном понимания игры.

## Stories

### STORY-004-001: Утренняя сводка
**User Story**: As a игрок, I want to видеть угрозы утром so that планировать день.
**Acceptance Criteria**:
- [ ] Сводка показывает важные изменения.
- [ ] Сводка выделяет высокую жару и угрозы врагов.
- [ ] Сводка показывает состояние бригад.
**Size**: S

### STORY-004-002: Вечерний отчет
**User Story**: As a игрок, I want to понимать последствия so that корректировать стратегию.
**Acceptance Criteria**:
- [ ] Каждый приказ имеет отдельную запись.
- [ ] Запись содержит итог, текст, эффекты.
- [ ] Отчет показывает действия врагов и полиции.
**Size**: M

### STORY-004-003: Простая карта
**User Story**: As a игрок, I want to видеть карту so that быстро читать состояние районов.
**Acceptance Criteria**:
- [ ] Консольная схема или 2D-блоки показывают 4 района.
- [ ] Видны владелец, доход, защита, жара.
- [ ] Цвет или метка показывает владельца.
**Size**: M
