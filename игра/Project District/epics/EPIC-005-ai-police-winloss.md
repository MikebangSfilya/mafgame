# EPIC-005: Враги, полиция, победа и поражение

## Description

Добавить системное давление мира: враги делают понятные действия, полиция реагирует на жару, игра умеет завершаться.

## Stories

### STORY-005-001: Rule-based действия врагов
**User Story**: As a игрок, I want to видеть действия врагов so that чувствовать борьбу за районы.
**Acceptance Criteria**:
- [ ] Каждая банда выбирает одно действие.
- [ ] Выбор основан на деньгах, угрозах, соседях, жаре или мести.
- [ ] Отчет объясняет мотивацию.
**Size**: M

### STORY-005-002: Жара полиции и рейды
**User Story**: As a игрок, I want to бояться жары so that насилие имело цену.
**Acceptance Criteria**:
- [ ] Приказы могут повышать или снижать жару.
- [ ] Высокая жара может вызвать рейд.
- [ ] Рейд снижает доход, наносит потери или арестовывает бригаду.
**Size**: M

### STORY-005-003: Условия победы
**User Story**: As a игрок, I want to иметь цель so that стратегия имела направление.
**Acceptance Criteria**:
- [ ] Победа за 3 района 5 дней подряд.
- [ ] Победа за уничтожение влияния конкурентов.
- [ ] Победа показывается отдельным отчетом.
**Size**: S

### STORY-005-004: Условия поражения
**User Story**: As a игрок, I want to понимать провал so that риск был реальным.
**Acceptance Criteria**:
- [ ] Поражение при потере всех районов.
- [ ] Поражение при общей жаре 10.
- [ ] Поражение объясняет главную причину.
**Size**: S
