# EPIC-003: Приказы и расчет исходов

## Description

Реализовать 5 приказов, качественную оценку риска и расчет пяти уровней исходов.

## Stories

### STORY-003-001: Выбор приказа
**User Story**: As a игрок, I want to выбрать бригаду, приказ и цель so that выразить намерение.
**Acceptance Criteria**:
- [ ] Доступны 5 приказов.
- [ ] Невалидные цели недоступны.
- [ ] Бригада получает не более одного приказа.
**Size**: M

### STORY-003-002: Предварительная оценка
**User Story**: As a игрок, I want to видеть риск до подтверждения so that принимать ответственность за решение.
**Acceptance Criteria**:
- [ ] Показываются шансы, риск полиции, риск самовольства.
- [ ] Оценка качественная, без процентов.
- [ ] Оценка меняется от параметров бригады и района.
**Size**: M

### STORY-003-003: Resolver приказов
**User Story**: As a игрок, I want to получать разные исходы so that игра не была бинарной.
**Acceptance Criteria**:
- [ ] Каждый приказ имеет 5 возможных исходов.
- [ ] Успех, риск и самовольство считаются отдельно.
- [ ] Эффекты применяются к состоянию мира.
**Size**: L

### STORY-003-004: Самовольство исполнителей
**User Story**: As a игрок, I want to видеть характер бригад so that они не были просто цифрами.
**Acceptance Criteria**:
- [ ] Низкая лояльность повышает риск отклонения.
- [ ] Высокая жестокость повышает риск шума.
- [ ] Отклонение всегда объясняется в отчете.
**Size**: M
