# REQ-004: Расчет пяти уровней исходов

Система MUST рассчитывать результат приказа по пяти уровням: полный успех, успех с осложнением, частичный успех, провал, катастрофа.

## User Story

As a игрок, I want to получать не бинарные результаты so that каждый день создавал новые компромиссы.

## Acceptance Criteria

- [ ] Каждый приказ возвращает один из пяти уровней исхода.
- [ ] Расчет учитывает параметры бригады, района, владельца, тип приказа и случайность.
- [ ] Риск полиции считается отдельно от шанса успеха.
- [ ] Риск самовольства считается отдельно от шанса успеха.
- [ ] Катастрофа имеет заметную цену: потери, арест, рост жары, война или потеря контроля.
- [ ] Успех с осложнением дает полезный эффект и негативное последствие.
