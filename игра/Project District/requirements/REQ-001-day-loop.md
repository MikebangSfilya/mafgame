# REQ-001: Дневной игровой цикл

Система MUST поддерживать полный цикл одного игрового дня: утренняя сводка, выбор приказов, автоматическое исполнение, вечерний отчет, реакция мира, переход к следующему дню.

## User Story

As a игрок, I want to проживать день за днем so that видеть последствия своих приказов и корректировать стратегию.

## Acceptance Criteria

- [ ] Игрок может начать День 1 с фиксированного стартового состояния.
- [ ] Игрок может назначить приказы всем свободным бригадам.
- [ ] После подтверждения дня система рассчитывает все приказы без ручного вмешательства.
- [ ] После расчета показывается вечерний отчет.
- [ ] Игра может перейти к следующему дню.
- [ ] Сессия может длиться 20 дней, если не выполнены условия победы или поражения.
