# ADR-001: Пошаговый локальный движок

## Context

Игра строится вокруг дневного цикла. Игрок планирует, затем весь мир исполняется автоматически.

## Decision

Использовать локальный turn engine, который пакетно рассчитывает один день и возвращает список событий и отчетов.

## Alternatives

| Option | Pros | Cons |
|--------|------|------|
| Real-time simulation | Живее выглядит | Раздувает сложность и ломает фокус |
| Пошаговый пакетный расчет | Простая проверка ядра | Меньше ощущения живого мира |
| Настольная ручная модель | Быстрее балансировать | Не дает полноценного прототипа |

## Consequences

- Все правила проще тестировать.
- UI может быть консольным.
- Отчеты становятся естественным результатом движка.
