# ADR-002: Фиксированная модель данных 0.1

## Context

Проект легко раздуть картой, типами юнитов и экономикой.

## Decision

В версии 0.1 использовать фиксированную карту из 4 районов, 3 банд и 2 бригад. Параметры вынести в данные, но не строить универсальный редактор.

## Alternatives

| Option | Pros | Cons |
|--------|------|------|
| Hardcode everything | Быстро | Сложно балансировать |
| Fixed config | Быстро и балансируемо | Не универсально |
| Full content editor | Гибко | Слишком дорого для 0.1 |

## Consequences

- Баланс можно менять быстро.
- Структура остается понятной.
- Расширение карты откладывается до проверки 4 районов.
