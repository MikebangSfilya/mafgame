# ADR-004: Отчеты как главный интерфейс

## Context

В игре про непрямое управление игрок узнает мир через последствия.

## Decision

Считать вечерний отчет самым важным экраном. Карта помогает ориентироваться, но доверие к игре строится через текст и эффекты.

## Alternatives

| Option | Pros | Cons |
|--------|------|------|
| Карта в центре | Быстро читается | Может увести в тактическую игру |
| Отчеты в центре | Поддерживает фантазию главаря | Требует хороших шаблонов текста |
| Таблицы параметров | Удобно балансировать | Сухо для игрока |

## Consequences

- Каждый resolver должен возвращать события, а не только изменять числа.
- Тексты отчетов являются частью механики.
